# Australian Rental Market Analysis 2026

## Overview

This project explores rental pricing patterns across Australia using a dataset of 6,449 rental property listings from 2026. The objective was to identify how location, property type, and bedroom count influence rental prices while developing an interactive dashboard to support data-driven decision making.

The project follows a complete analytics workflow, from data cleaning and preparation through to exploratory analysis, insight generation, and dashboard development.

---

## Tools

- Microsoft Excel
- Power BI

---

## Analysis Focus

The analysis was designed to answer several key questions:

- Which Australian states have the highest average rental prices?
- How does rental pricing vary across property types?
- What is the relationship between bedroom count and rental price?
- Which property categories provide the best rental value?

To support fair comparison across different property sizes, a custom **Rent per Bedroom** metric was created and analysed.

---

## Key Findings

- NSW recorded the highest average rental price among all states in the dataset.
- Rental prices generally increased with bedroom count, although the relationship was not strictly proportional.
- Property type had a significant influence on rental pricing, with acreage and semi-rural properties achieving the highest average rents.
- Rent-per-bedroom analysis revealed notable differences in rental value across property categories, highlighting that higher total rent does not always represent lower value.

---

## Dashboard Features

The project includes two interactive Power BI dashboards:

### 1. Executive Overview

Designed to provide a high-level summary of Australia's rental market, including state-level pricing comparisons, property type analysis, bedroom-based pricing trends, and identification of premium rental locations.

### 2. Rental Value & Affordability Analysis

Designed to evaluate rental value using rent-per-bedroom metrics, highlighting affordability differences across states, property types, bedroom counts, and suburbs.

Both dashboards support interactive filtering by state, property type, and bedroom count, allowing users to explore market trends from multiple perspectives.
